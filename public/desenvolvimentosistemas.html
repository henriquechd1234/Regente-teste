Curso Técnico em Desenvolvimento de Sistemas

Formação profissional para atuar no desenvolvimento de software e soluções tecnológicas

Galeria do Curso

Sobre o Curso

O curso técnico em Desenvolvimento de Sistemas prepara profissionais para atuar no desenvolvimento de software, aplicações web e mobile, banco de dados e soluções tecnológicas para diversos setores.

Duração: 34 meses
Modalidade: Presencial
Carga Horária: 1200 horas
Investimento: R$ 0/mês

Mercado de Trabalho

O técnico em Desenvolvimento de Sistemas pode atuar em diversas áreas como:

  • Desenvolvimento de software
  • Desenvolvimento web e mobile
  • Banco de dados
  • Testes de software
  • Suporte técnico
  • Consultoria em TI

  • Projeto Integrador II
